# OSINT Plugin for ATAK

An ATAK plugin developed for Open Source Intelligence (OSINT) operations in Arizona.

## Overview

This plugin integrates OSINT capabilities directly into ATAK, allowing operators to gather and visualize open-source intelligence data without leaving the ATAK environment.

## Features

- OSINT data aggregation and display within ATAK
- Integration with open-source intelligence sources
- Arizona-focused data overlays

## Installation

> ⚠️ **Work in Progress** — Installation instructions will be added as the plugin matures.

1. Download the latest `.apk` release from the [Releases](../../../releases) page
2. Sideload the APK onto your ATAK-enabled Android device
3. Enable the plugin within ATAK under **Settings > Manage Plugins**
